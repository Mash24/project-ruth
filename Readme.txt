# A Boa Vida - Portfolio Website

## Overview
A Boa Vida is a portfolio website designed to showcase products and services, allowing potential customers to view offerings and contact the website owner for further assistance. This project is built using HTML, CSS, and JavaScript, with Bootstrap as the primary framework for responsive design.

## Features
- **Responsive Design**: The website is fully responsive, ensuring a seamless experience across various devices and screen sizes.
- **Portfolio Showcase**: A dedicated section to display products and services offered.
- **Contact Form**: Users can fill out a contact form to reach out for inquiries or assistance.
- **Navigation Menu**: Easy navigation through different sections of the website.

## Technologies Used
- **HTML**: For structuring the content of the website.
- **CSS**: For styling the website, utilizing Bootstrap for layout and design.
- **JavaScript**: For interactive elements and form handling.
- **Bootstrap**: A front-end framework used to create a responsive layout and design.

## Installation
1. Clone the repository:
   ```bash
   git clone git remote add origin https://github.com/Mash24/project-ruth.git

   ```
2. Navigate to the project directory:
   ```bash
   cd a-boa-vida
   ```
3. Open `index.html` in your web browser to view the website.
